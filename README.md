## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```

---

### Setup Netlify

```bash
# install netlify-cli
$ npm install -g netlify-cli
```

### Add Module 

```bash
$ npm install @nuxtjs/sitemap
```
